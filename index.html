Enjoy Reading

Enjoy Reading is a Firefox extension to improve websites readability and get the best online reading experience.

↺Get it now Works with Firefox 4.0 - 4.2a1pre

Description

Enjoy Reading is heavily based on the old Readability open source bookmarklet.

The only difference for now (this is going to change) is that there is no more server calls to the readability server, the add-on embeds all the logic !

Everything happens on the client side, no more tracking, and way quicker !

Oh and by the way, this add-on is and will stay open-source and free.

↺Get it now Works with Firefox 4.0 - 4.2a1pre

Usage

After installing the add on you should have a new icon in your toolbar menu. Just click it to activate the add-on (and click again to return to the original page).
